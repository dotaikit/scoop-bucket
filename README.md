# dotaikit Scoop Bucket

Scoop manifests for dotaikit tools.

## Install

```powershell
scoop bucket add dotaikit https://github.com/dotaikit/scoop-bucket
scoop install acm
```

## Upgrade

```powershell
scoop update
scoop update acm
```

Manifests live in `bucket/`. The `acm` manifest is updated automatically by the `dotaikit/acm` release workflow.
